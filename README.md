# Digital Diary

A simple and user-friendly digital diary web application built using HTML, CSS, and JavaScript. This project allows users to write, save, and manage their daily thoughts directly in the browser without requiring any backend or database.

## Features
- Add and save diary entries
- Automatically records date and time for each entry
- Data is stored in browser local storage
- Delete entries when no longer needed
- Clean and simple user interface

## Project Structure
digital-diary/
│── index.html  
│── style.css  
│── script.js  

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/digital-diary.git  
2. Open the project folder  
3. Open the index.html file in any web browser  

## How It Works
- The user writes an entry in the text area and clicks the save button  
- The entry is stored in the browser using local storage  
- Saved entries are displayed on the page along with date and time  
- Users can delete entries, which updates the stored data instantly  

## Technologies Used
- HTML for structure  
- CSS for styling and layout  
- JavaScript for functionality  
- Local Storage for saving data
