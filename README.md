# Mini Projects Collection: Digital Diary & XOX Game

## Description


1. **Digital Diary** – A web-based application for writing and managing personal diary entries.
2. **XOX Game (Tic Tac Toe)** – A simple two-player game built using basic programming logic.

These projects demonstrate fundamental concepts of web development and programming.

---

## 📘 Digital Diary

### Features

* Write and save diary entries
* Clean and simple user interface
* Responsive layout

### Technologies Used

* HTML
* CSS
* JavaScript

### Project Structure

```
Digital-Diary/
│── index.html
│── style.css
│── script.js
```

### How to Run

1. Open the project folder
2. Open `index.html` in any browser

### Future Improvements

## ❌⭕ XOX Game (Tic Tac Toe)

### Features

* Two-player gameplay
* Input-based system
* Win/draw detection

### Technologies Used

* C++ (or HTML/CSS/JavaScript for web version)

### Project Structure

```
XOX-Game/
│── main.cpp
```

### How to Run (C++ Version)

1. Open the file in a C++ compiler
2. Compile the code
3. Run the program
4. Enter positions to play

### How to Play

* The board has positions from 1 to 9
* Players take turns marking X and O
* First to align three marks wins

### Future Improvements

* Add AI opponent
* Improve user interface
* Add score tracking
* Restart option
